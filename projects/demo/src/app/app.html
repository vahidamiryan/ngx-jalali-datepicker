ngx-jalali-datepicker

An Angular date picker built on signals, an extensible adapter architecture, and a headless core — Jalali · Gregorian · Hijri, shipping a tiny bundle with full tree-shaking.

⚡ Signals & Zoneless 🌳 Tree-shakeable 🗓 Three calendars ⌨ Full keyboard nav 🌙 Dark mode 🔌 Headless core
Try it live GitHub

$ npm install @vahidamirian/ngx-jalali-datepicker
