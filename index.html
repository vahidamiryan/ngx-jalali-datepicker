Shared headless core

All calendar math, adapters, selection and view-building live in @vahidamirian/datepicker-core — pure TypeScript, zero framework deps. Fix a bug once, both frameworks get it.

Jalali · Gregorian · Hijri

Battle-tested conversion math verified against the platform Intl calendars, with dual-script display and a pluggable adapter layer for any calendar.

Angular & Vue, identical behavior

@vahidamirian/ngx-jalali-datepicker (signals, zoneless, OnPush) and @vahidamirian/vue-datepicker (Composition API, v-model, scoped slots) render the same UI from the same core.

Fast by construction

Every per-day flag is precomputed once; templates read only booleans. Numeric yyyymmdd keys, no per-cell string allocations.
